# Sport Performance App

スポーツパフォーマンス Top 10 を表示する TypeScript / React アプリ（ライトカラーパターン）。

## セットアップ

```bash
npm install
```

## 開発

```bash
npm run dev
```

## ビルド

```bash
npm run build
```

## 含まれるスポーツ

- バスケットボール（NBA MVP 2回以上）
- 水泳（オリンピック金メダル 3個以上）
- 短距離走（100m/200m/4x100m 金メダル 3個以上）
- 長距離走（5000m/10000m/マラソン 金メダル 3個以上）
- バレーボール（室内＋ビーチ 金メダル 3個以上）
- バドミントン（金メダル 2個以上）
- 卓球（金メダル 3個以上）
